# 🃏 Super Trunfo em C

Um jogo estilo “Super Trunfo” implementado em C que permite comparar cartas com atributos e ganhar do oponente.  
Desenvolvido como exercício de lógica, módulos em C e interação via terminal.

## 🚀 Tecnologias  
- Linguagem: C  
- Compilador: gcc (ou equivalente)
- Ambiente: terminal / console
